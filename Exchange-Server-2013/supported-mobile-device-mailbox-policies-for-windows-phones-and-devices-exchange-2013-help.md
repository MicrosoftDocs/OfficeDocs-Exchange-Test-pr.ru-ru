# Поддерживаемые политики почтовых ящиков мобильных устройств для Windows Phone и прочих устройств

 

_**Применимо к:** Exchange Server 2013_

_**Последнее изменение раздела:** 2015-03-09_

С выпуском Windows Phone 8, Windows 8 и Windows RT появился ряд устройств, поддерживающих Exchange ActiveSync и политики почтовых ящиков мобильных устройств. Операционная система каждого из устройств поддерживает определенный набор параметров политик почтовых ящиков мобильных устройств.

## Матрица совместимости для клиентов Windows и Exchange Server

В приведенных ниже таблицах перечислены совместимые параметры политик почтовых ящиков мобильных устройств для различных версий Windows Phone, Windows 8, Windows RT и сервера Exchange Server.
